## üìå How to Apply the Save Files

### Method 1 ‚Äî Copy & Paste Content

1. Close **Megabonk**
2. Press `Win + R` and type: `%appdata%`
3. Navigate to: `..\LocalLow\Ved\Megabonk`
4. Open `progression.json` and/or `stats.json`
5. **Replace the entire content** with the content from this repository
6. Save the files
7. Launch the game

### Method 2 ‚Äî Replace Files Directly

1. Close **Megabonk**
2. Press `Win + R` and type: `%appdata%`
3. Navigate to: `..\LocalLow\Ved\Megabonk`
4. Download the files from this repository
5. **Overwrite** the existing files:
- `progression.json`
- `stats.json`
6. Launch the game
